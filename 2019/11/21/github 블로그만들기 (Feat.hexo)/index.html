github 블로그만들기 (Feat. hexo)

Nov 21, 2019

기억력이 좋지 못해서, 무언가 알게된 지식을 기존엔 evernote에 저장을 했었다.
차곡차곡 문서가 쌓이다보니…. 뭘 작성한지도 작성한걸 찾지도 못하는 나를 발견하여
느지막이 github 블로그를 만들어 보기로 결심.
이왕하는거 hexo를 이용해보기로 했다.

github repository 생성

  • repository 명칭은 USERNAME.github.io 로 생성 (thinkub.github.io)
  • Description 은 마음데로
  • public 설정하고
  • README 파일은 생성한다.

hexo

  • git cron
    먼저 생성한 blog용 git repository를 cron한다

  • hexo 설치

    • cron 한 경로에 blog 라는 폴더로 hexo 를 init

    • 첨부된 이미지 처럼 blog 라는 경로가 생김

    • local 에서 확인하기

  • theme 적용
    여기 에서 원하는 걸 고르면된다.
    나는 clean-blog 를 사용하기로 했음

    • 선택한 theme repository git cron

    • theme 적용

  • github 동기화

    • /blog/_config.yml 파일을 보면 Deploy 설정하는 곳이 있다 여기에 설정

    • public 파일 변환

    • git에 올리기
